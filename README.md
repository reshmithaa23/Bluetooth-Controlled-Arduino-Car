# Bluetooth Controlled Arduino Car 

##  Description
This project is a Bluetooth-controlled car using Arduino. It can be controlled wirelessly using a mobile application.

##  Components Used
- Arduino Uno
- HC-05 Bluetooth Module
- Motor Driver (L298N)
- DC Motors
- Battery
- Jumper wires

##  Working
The mobile app sends commands via Bluetooth.
The HC-05 module receives them and sends to Arduino.
Arduino controls the motors based on commands.

##  Features
- Forward
- Backward
- Left
- Right
- Stop
